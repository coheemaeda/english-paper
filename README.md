# 海外ニュース情報収集（Google Scholar × Gemini）

Google Scholar の通知メールから論文を選定し、Gemini AI で要約・レポート化してスプレッドシートに保存・メール通知する Google Apps Script。

## 概要

1. **AI編集長（選定）**: Scholar 通知メールから「日本の少子化・子育て政策」に関連する最も価値のある論文を最大1本選定
2. **全文取得**: 選定された論文の本文を取得
3. **AI記者（執筆）**: 3行要約・500文字要約・エビデンス・信憑性評価・日本への示唆・SNS投稿案を生成
4. **保存・通知**: スプレッドシートに記録し、HTMLメールで通知

## 前提条件

- Google アカウント
- Google Scholar のメール通知（ラベル `10_scholar` などで管理）
- Gemini API キー（[Google AI Studio](https://aistudio.google.com) で無料取得可能）

## セットアップ

### 1. スプレッドシートの準備

1. [Google スプレッドシート](https://sheets.google.com) で新規作成
2. 拡張機能 > Apps Script でプロジェクトを開く
3. `enlish-paper.gs` の内容を貼り付け

### 2. スクリプトプロパティの設定（必須）

個人情報はコードに含めず、GAS のスクリプトプロパティで管理します。

1. 左サイドバー **プロジェクトの設定**（歯車アイコン）をクリック
2. **スクリプト プロパティ** までスクロール
3. **プロパティを追加** で以下を設定：

| プロパティ | 値 |
|-----------|-----|
| `GEMINI_API_KEY` | [Google AI Studio](https://aistudio.google.com) で取得した API キー |
| `EMAIL_RECIPIENT` | 通知を受け取るメールアドレス |

### 3. Gmail ラベルの設定

コード内の `GMAIL_QUERY` を、あなたの Scholar 通知ラベルに合わせて変更してください。

```javascript
const GMAIL_QUERY = 'label:10_scholar label:unread';  // 例: 10_scholar ラベル
```

### 4. トリガーの設定

1. 左サイドバー **トリガー**（時計アイコン）をクリック
2. **トリガーを追加** → 関数 `main` を選択
3. イベントの種類: **時間主導型**、間隔: **日付ベースのタイマー**（例: 毎日午前9時）
4. **通知先アカウント**（`EMAIL_RECIPIENT` と同じアカウント）でトリガーを設定すること（送信元＝実行アカウント）

## Gemini API キーの取得方法

1. [Google AI Studio](https://aistudio.google.com) にアクセス
2. Google アカウントでログイン
3. 左メニュー「API Keys」→「Create API key」
4. 表示されたキーをコピー（**一度しか表示されない**ため必ず保存）

## 主な関数

| 関数 | 説明 |
|------|------|
| `main` | 本番処理。未読 Scholar メールを処理し、選定→取得→分析→保存→通知 |
| `selectBestPapers` | AI編集長。メール本文から論文URLを最大1本選定 |
| `analyzeWithGemini` | AI記者。論文全文を分析しレポート生成 |
| `sendRichEmail` | HTML形式でメール通知 |

## スプレッドシートの列構成

| 列 | 内容 |
|----|------|
| 1 | 処理日時 |
| 2 | タイトル |
| 3 | 3行要約 |
| 4 | 500文字要約 |
| 5 | KEYデータ/エビデンス |
| 6 | 信憑性評価 |
| 7 | 日本への示唆 |
| 8 | SNS投稿案（3パターン） |
| 9 | 論文URL |
| 10 | 著者 |
| 11 | 公開日 |

## 注意事項

- スクリプトプロパティは GAS プロジェクトに紐づき、GitHub には含まれません
- 初回実行時に Gmail 送信・スプレッドシート編集の権限承認が必要です
- 論文取得の連続アクセス防止のため、1本あたり10秒の休憩を入れています
- Gemini API には無料枠があります（利用制限あり）
